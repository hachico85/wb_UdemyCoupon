# 要件定義書：Udemy独自クーポン置き場

## 1. プロジェクト概要
Udemy講師としての独自のクーポンを一覧で掲載するWebサイトを作成する。
メルマガ登録者やSNSプロフィールからの訪問者を対象とし、常設のクーポン配布場所として機能させる。

## 2. ターゲットユーザー
- メルマガ登録者
- SNS（Twitter/X, LinkedInなど）のプロフィール経由の訪問者
- 既存の受講生
- **デザインの方向性**: スタイリッシュ、センスの良さを感じるデザイン

## 3. 目的
- 受講生に最安値や特典付きでコースを提供し、満足度を高める
- 新規コースや既存コースの販売促進
- 「ここに来ればいつでもお得なクーポンがある」という認知を獲得する
- **運用**: 月に1回程度のクーポン更新が発生するため、メンテナンスしやすい構成にする

## 4. 機能要件
### 必須機能
- **クーポン一覧表示**: 6つのコース（2025/11時点）をカード形式で表示
- **コース情報**:
    - コースサムネイル画像
    - コースタイトル
    - 割引価格 / 割引率
    - クーポンコード（コピー機能があると望ましい）
    - コースへのリンクボタン
- **レスポンシブ対応**: スマホ、タブレット、PCで見やすく表示
- **OGP設定**: SNSでシェアされた際に魅力的なプレビューを表示（タイトル、説明、画像）
- **メンテナンス性**: HTML内の特定箇所を書き換えるだけで更新できるようにする（またはJSON分離）

### ホスティング
- **GitHub Pages**: 静的サイトホスティングとして利用。セキュリティ上の懸念事項（個人情報や秘密鍵）は含まないため問題なし。

## 5. 非機能要件
- **パフォーマンス**: 画像の最適化などにより高速に表示する
- **デザイン**: 信頼感があり、かつ「お得感」が伝わるデザイン
- **保守性**: HTML/CSSベースで、クーポンの差し替えが容易であること

## 6. ページ構成
- **ヘッダー**: サイトタイトル、簡単な挨拶/説明
- **メインコンテンツ**: クーポン一覧エリア
- **フッター**: 著作権表示、SNSリンク、メルマガ登録リンク（あれば）
